My name is Robert and this is my website. This space allows we to host my own privite Computer Science projects. I have always loved computer technology and I am finally able to explore my hobby a little deeper. I will showcase my various projects with the hope to inspire others to crate.


I coded this entire website by hand, using examples and lessons from freecodecamp.org. The main pages are coded using basic HTML and CSS. No JS nor any other scripts were used.


We are currently running through AWS on Debian 10. I am able to quickly deploy and maintain remote webservices. My primary expertise in remote devops is with Linux family of operating systems, mostly Debian.


From this Virtual Private Server platform I will be able to deploy several web technologies, building a resume of services I am able to work with.The 4th quarter of 2020 will see the deployment of a few Django Webframework projects.


By creating this site with just CSS and HTML allows me to create the site the way I want to without having to worry about being in the confines a specific framework. The only thing I need is text editor and a terminal. Nano has become my new best friend.
